1. 让你做自我介绍。
2. 你是否仍在实习？实习中 AI 产品面向的问答场景/问题类型有哪些？
3. 文档中有图片等异构数据时：从解析到可检索的处理链路怎么做？只返回文本会不会丢信息？
4. 图文对齐是怎么建立的（人工标注/弱监督等）？量大时如何控制成本？
5. 异构数据解析后如何保障回答准确：如何发现错误、定位问题、设计验证方案？
6. 用另一个大模型做评测时：如何构造问题、对齐对比口径、衡量一致性与正确率？
7. 你们做过哪些图像/多模态增强？如何控制幻觉与文档外生成？
8. 用户问到文档外问题时怎么处理（fallback/澄清/检索扩展等）？
9. 这些流程是否有统一编排框架？用什么实现（工作流/Agent 框架等）？
10. 简历中的“检索召回率/忠实度”等指标怎么定义、分子分母怎么取？
11. 设备故障叙述这类复杂文本：模型如何理解？术语解释/信息抽取怎么做？
12. LoRA 除了省参数/显存外还有什么收益？你做的 LoRA 工作是否落地过？
13. 语言栈：主要用 Python 还是 Java？是否了解 Java 多线程/类加载等基础？
14. 你用过 LangChain 吗？如果要缓解幻觉，你会用哪些思路（非 LangChain 也可）？
15. 让你做一个“查询天气”的对话 AI：顶层设计与端到端步骤怎么拆？
16. 天气系统如何处理地理知识（城市-国家映射等）并对接天气 API？
17. 若查询后要继续推荐/下游服务联动：如何让模型按顺序调用多个服务？服务多时如何工程化？
18. LangChain 和 LlamaIndex 的定位差异与适用场景？
19. 怎么测试这类 AI 系统：评测数据集如何构建、测试分哪些层次、验证哪些能力？
20. 设计“公司周边点奶茶”的 Agent：定位/门店查询/下单 API 如何串联？常见失败点与解决？
21. 如果定位坐标被模型“编错”导致找店错误：如何纠正、让模型承认不确定、并做可恢复处理？

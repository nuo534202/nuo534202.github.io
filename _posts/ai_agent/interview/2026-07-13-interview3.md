1. 实习经历深挖（做了什么、难点、产出）。
2. 项目经历深挖（架构、链路、你负责的部分）。
3. 你的检索是否基于向量？搭一个完整 RAG 需要哪些核心组件？
4. 文档上传与分块如何实现（chunk 策略、清洗、索引构建）？
5. 向量检索单次耗时是多少？是否引入 rerank，怎么评估收益？
6. 如何评估召回内容与问题匹配度（离线/在线指标与标注）？
7. 你如何定义/理解 Agent？核心模块有哪些？
8. 是否了解 ReAct 等 Agent 范式？在项目里怎么用？
9. Agent 循环一般多少步能收敛？到最大步数仍失败时怎么处理？
10. Agent 接入了哪些工具？工具接口与权限怎么治理？
11. 如何“约定并约束”模型做正确的工具调用（schema、提示、校验、回退）？
12. 推流是否用 SSE？为什么选 SSE？
13. 编排里是否有 Plan 阶段？Plan-Execute 如何实现？
14. 协议层面对比 SSE / WebSocket / HTTP：差异、适用场景与取舍。
15. 为什么限制每个用户只能上传一个文件（产品/工程约束）？
16. 图像识别怎么做？为什么不用多模态大模型，而选传统识别方案？
17. 项目用到哪些大语言模型？选型原因是什么？
18. Go 并发与其他语言并发模型的关键差异是什么？
19. Goroutine 的基本原理（调度、栈、M/P/G 等理解）。
20. 你如何理解“锁”：它解决什么问题？
21. 业务中常用哪些锁？还了解哪些后端常见同步原语？
22. 多机/多进程场景会涉及哪些锁？分布式锁怎么做、风险点在哪？
23. 你如何理解 MCP、Function Calling、A2A：分别解决什么问题？
24. 项目接入了哪些 MCP 服务？是否手写过本地 MCP Server？
25. 日常开发会用哪些 AI 工具（IDE/模型/插件）？
26. 使用这些工具的经验：是否写过自定义 command/skill？
27. Claude Code 的实现原理/关键设计点（如记忆、上下文管理等）。
28. 是否了解 OpenClaw？对其实现思路的理解。
